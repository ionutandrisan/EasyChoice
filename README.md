## Rezumat  
  

   &nbsp;&nbsp;&nbsp;&nbsp;Subiectul lucrării de diplomă este constituit de implementarea unei aplicațieiii web de rezervare a locuințieielor numită SmartChoice. Această aplicațieiie oferă majoritatea intrumentelor necesare activitățieiii de rezervare și închiriere a locuințelor, cu scopul de a petrece o vacanțieiă sau a petrece timp de calitate.  
   &nbsp;&nbsp;&nbsp;&nbsp;Crearea și dezvoltarea acestei aplicații a implicat atât crearea unei interfețe web cu care utilizatorii vor avea contact, cât și dezvoltarea serviciilor din spatele aplicațieiiei datorită cărora este posibil accesul la o serie de funcționalități. 
   &nbsp;&nbsp;&nbsp;&nbsp;SmartChoice face partea din categoria site-urilor de călătorii pentru rezervarea locuințieielor. Scopul acestei aplicațieiii este de a găsi locul perfect pentru o vacanță sau de a obține un venit în plus, prin închirierea unui spațieiiu dețieiinut.  
     &nbsp;&nbsp;&nbsp;&nbsp;Pentru implementarea acestui site a fost nevoie de o dezvoltarea atât pe partea de client, cât și una pe partea server. Pentru implementarea acestor componente am ales să folosesc următoarele tehnologii:

* ReactJS pentru implementarea interfețeiei cu clientul. 
* Spring Boot, împreună cu Spring Security, pentru implementarea server-ului de servicii și a securității aplicației.  
* Oracle SQL pentru stocarea și manipularea informațiilor.  

&nbsp;&nbsp;&nbsp;&nbsp; SmartChoice dispune de următoarele funcționalități:
* Crearea unui cont și adăugarea unei fotografii de profil
* Adăugarea unui nou spațiu pentru închiriat
* Afișarea tuturor spațiilor adăugate de un utilizator
* Căutarea tuturor spațiilor dintr-un oraș.
* Verificarea valabilității unei locuințe
* Rezervarea unui spațiu
* Scierea unei recenzii
* Anularea unei rezervări
* Notificarea prin email a unui propretar când o locuință care îi aparțieiine este rezervată
* Afiși unor grafice pe care proprietaul le poate analiza în determinarea profitului
